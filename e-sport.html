E-Sport Data Analytics

Infographic  •  Animation  •  Data Analytics

Getting Into the Game is an animated infographic about the esports industry I made during the 2018 Temple Analytics challenge. The challenge was to use data to give recommendations to media companies on how to engage with esports users. In order to create this project, I had to go through massive amounts of data to analyze patterns and create easy-to-understand infographics. While most contestants competed in teams, I did the project end-to-end by myself.

1st place in the design category of 6th Annual Comcast NBCUniversal Temple University Data Analytics Challenge
