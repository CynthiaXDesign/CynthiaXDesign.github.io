Solar System Sorting Game

P5JS  •  Creative Coding  •  Website Game

Let's re-create our solar system. People can drag the planets on the top to their corresponding circle to match the right order in the solar system. The planet will stay in place if it is placed in its right position. Now, play it and test you how much you know about the solar system.
Tips:
– The planets on the top are shuffled randomly each time to avoid people memorizing a fixed pairing.
– This website is compatible with a desktop or laptop environment.

The Farm Beats

P5JS  •  Creative Coding  •  Website Game

This virtual piano rendered on top of a beautful farm background can teach children how to play the piano and sing with friends. Try it and you will hear the melody "Old McDonald had a Farm."
This app is designed for an iPad so children can tap on the screen like a real piano.
Tips:
If you open it by desktop, please zoom out the screen, and then you will see it whole. Command + "-"

Winner of the 2019 AIGA Flux Student Design Competition, check it here: http://fluxdesigncompetition.org/winners.php
